# NightGuard Flashlight 隐私政策

**生效日期：** 2026年3月31日

---

## 1. 概述

欢迎使用 **NightGuard Flashlight**（以下简称“本应用”）。本隐私政策旨在说明本应用如何处理您的个人信息。

**请注意：本应用是一款纯粹的本地工具类应用。** 为了保障您的隐私安全，我们设计本应用时遵循了“最小化权限”原则。**本应用不收集、存储、传输或处理任何您的个人身份信息。**

## 2. 权限使用说明

本应用仅申请运行所必需的系统权限，且所有操作均在您的设备本地完成：

- **相机/闪光灯权限**：用于开启和控制手机的手电筒功能、SOS 信号闪烁及呼吸灯效果。
  - **用途**：实现应用的核心功能（照明和紧急信号）。
  - **数据流向**：权限仅用于控制硬件，**不涉及任何网络传输或数据上传**。

## 3. 数据收集与使用

本应用**不会**：

- 收集您的姓名、电子邮件、电话号码或其他个人身份信息。
- 访问您的通讯录、位置、相册或麦克风。
- 与任何第三方（包括广告商）共享您的数据。
- 在服务器端存储您的任何使用记录。

本应用**可能会收集**（可选，视具体代码实现而定）：

- **崩溃日志**：如果您开启了“自动发送错误报告”功能（通常在系统设置中），iOS 系统可能会向开发者发送匿名的崩溃日志，以帮助我们改进应用稳定性。这些日志不包含个人身份信息。

## 4. 第三方服务

本应用不包含任何第三方 SDK（如广告 SDK、分析 SDK 等）。所有功能均由本地代码实现。

## 5. 隐私政策的变更

我们可能会不时更新本隐私政策。更新后的隐私政策将在应用内或通过 GitHub 仓库发布。我们建议您定期查看本页面。
